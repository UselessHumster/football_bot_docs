# Начало работы

## Добавление бота в чат
Добавьте бота в чат команды:

Зайдите в меню управления чатом

![img.png](assets/chat.png)

Выберите пункт "Ещё"

![img_1.png](assets/chat_more.png)

Затем "Добавить участников"

![img_2.png](assets/chat_add_member.png)

Найдите бота в списке

![img_3.png](assets/chat_add_bot.png)

Нажмите "Добавить"

![img_4.png](assets/chat_add_confirm.png)

Затем ещё раз "Добавить"

![img_5.png](assets/chat_confirm2.png)

## Выдача прав
Выдайте боту права администратора

Зайдите в меню управления чатом

![img.png](assets/chat.png)

Выберите пункт "Управление"

![img_6.png](assets/chat_manage.png)

Выберите пункт "Администраторы"

![img_7.png](assets/chat_administrators.png)

Нажмите "Добавить администратора"

![img_8.png](assets/chat_add_admin.png)

Затем выберите бота

![img_9.png](assets/chat_add_bot_as_admin.png)

Выставьте переключатели в таком же порядке, как на скриншоте и нажмите "Сохранить"

![img_11.png](assets/chat_confirm_bot_as_admin.png)

## Получения ID чата
Запустите команду `/chat@ReddysAdmin_bot`

Бот даст понять реакцией, что команда сработала успешно

![img_14.png](assets/chat_command.png)

Напишите администрации, что все шаги сделаны
